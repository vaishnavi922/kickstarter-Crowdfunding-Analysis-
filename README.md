## Kickstarter Crowdfunding Analysis
# 📌 Project Overview
This project analyzes Kickstarter crowdfunding campaign data to identify key factors that influence project success and failure. 
The dataset was cleaned, transformed, and analyzed to uncover meaningful insights that can help in making data-driven decisions for future crowdfunding campaigns
# 🎯 Project Objectives
Analyze Kickstarter dataset to identify factors affecting campaign success and failure.
Examine the relationship between funding goal and pledged amount.
Evaluate the impact of campaign duration on success rate.
Perform category-wise and country-wise analysis to identify top-performing segments.
Analyze trends based on launch year.
Clean and preprocess raw data for accurate analysis.
Build an interactive dashboard for visualization and insights.
# 🛠️ Tools & Technologies Used
Microsoft Excel (Data Cleaning & Preprocessing)
SQL (Data Querying & Analysis)
Power BI (Dashboard & Visualization)
# 🔍 Key Findings
Projects with realistic funding goals have higher success rates.
Short-to-medium duration campaigns tend to perform better.
Certain categories consistently show higher success percentages.
Campaign trends show growth over the years in crowdfunding participation.
# 🚀 Conclusion
This project demonstrates how data analysis and visualization techniques can help understand crowdfunding trends and improve strategic decision-making for campaign success.
